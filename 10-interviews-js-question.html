Điều gì về JS khiến nó khác biệt hơn so với cái ngôn ngữ khác? Đây là một vài câu hỏi sẽ giúp bạn khám phá ra những điều thực sự quan trọng: 1. Paradigms là gì ? Bạn có thể kể tên 2 paradigms lập trình quan trọng cho các LTV JS? Paradigms là một mô hình lập trình, bạn có thể hiểu đơn giản hơn paradigms là 1 style, 1 hướng để code. JS là một ngôn ngữ rất rất tự do. Nó hỗ trợ lập trình thủ tục, bắt buộc, khô khan với OOP. Nhưng nó cũng hỗ trợ cả function programming (lập trình hàm). JS hỗ trợ OOP với prototypal inheritance (sự kế thừa mẫu object) Điểm cộng : - Prototypal inheritance (also: prototypes, OLOO). - Functional programming (also: closures, first class functions, lambdas).
